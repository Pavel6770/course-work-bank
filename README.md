# Магазин: классы Product и Category

Проект демонстрирует объектно-ориентированное программирование на Python на примере системы управления товарами интернет-магазина.

---

## 📦 Классы и функциональность

### Product
- `name` — название товара
- `description` — описание
- `price` — цена (приватный атрибут `__price` с геттером/сеттером)
- `quantity` — количество
- Валидация цены (не может быть ≤ 0)
- Класс-метод `new_product` — создаёт товар из словаря, объединяет дубли по имени (берёт максимальную цену и складывает количество)
- Магические методы: `__str__`, `__add__`, `__repr__`

### Category
- `name` — название категории
- `description` — описание
- `_products` — приватный список товаров
- Метод `add_product` для добавления товаров
- Геттер `products`, возвращающий строковое представление всех товаров
- Атрибуты класса: `category_count`, `product_count`
- `__str__` и `__iter__`

### Ограничения
- **Сложение (`__add__`)**: можно складывать только объекты одного класса (проверка через `type()`).
- **Добавление в категорию (`add_product`)**: можно добавлять только объекты Product или его наследников (проверка через `isinstance()`).
